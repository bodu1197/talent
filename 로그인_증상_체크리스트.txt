============================================
로그인 문제 증상 체크리스트
============================================

다음 중 어떤 증상인가요? 해당하는 번호를 알려주세요:

1. 로그인 버튼을 클릭해도 아무 반응 없음

2. "로그인 중..." 표시 후 계속 멈춤 (무한 로딩)

3. 로그인 성공 → 홈페이지(/)로 이동 → 빈 화면 표시

4. 로그인 성공 → 홈페이지(/)로 이동 → 무한 로딩

5. 로그인 시도 → 빨간색 에러 메시지 표시
   (에러 메시지: _________________)

6. 로그인 성공 → 다른 페이지로 이동됨
   (어느 페이지: _________________)

7. F12 개발자 도구 Console에 빨간색 에러 표시
   (에러 내용: _________________)

8. 기타 (구체적으로: _________________)
