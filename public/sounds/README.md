# 알림음 파일

이 폴더에는 채팅 알림음 파일이 필요합니다.

## 필요한 파일

- `notification.mp3` - 채팅 메시지 알림음

## 추천 알림음 사이트

무료 알림음을 다운로드할 수 있는 사이트:

- https://pixabay.com/sound-effects/search/notification/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

## 파일 요구사항

- 파일명: notification.mp3
- 길이: 1-2초 권장
- 용량: 50KB 이하 권장
- 형식: MP3

## 임시 대체 방안

알림음 파일이 없어도 코드는 정상 작동하며, 파일이 없을 경우 알림음만 재생되지 않습니다.
